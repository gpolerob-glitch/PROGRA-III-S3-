Analizador de Texto
Descripción

Aplicación desarrollada en Python que permite analizar un texto ingresado manualmente o desde un archivo .txt. Genera estadísticas sobre las palabras y permite realizar consultas de frecuencia.

Funcionalidades
Cargar texto desde archivo TXT
Ingresar texto manualmente.
Normalización de texto.
Conteo total de palabras.
Conteo de palabras únicas.
Top 10 palabras más frecuentes.
Longitud promedio de palabras.
Consulta de frecuencia de palabras.
Manejo de errores y validaciones.
Tecnologías Utilizadas
Python 3
Tkinter
Programación Orientada a Objetos
Dictionaries
Sets
Excepciones
Ejecución
python main.py
